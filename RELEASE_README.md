# Video Motion Detection - v1.0.0

Stable release of the Streamlit application.

## Repository

https://github.com/trungtin-dinh/video_motion_detection

## Run locally

Create or activate a Python environment, then install the dependencies:

```bash
pip install -r requirements.txt
```

Launch the Streamlit app:

```bash
python -m streamlit run app_sl.py
```

## Contents

This release contains the source code of the application, the Streamlit entry point, the dependency file, the documentation, and the assets needed by the app when they are present in the repository.

## Notes

This is a source-code release for a Streamlit portfolio app. It is not a native desktop executable. The app can be used online through Streamlit Cloud when a public deployment is available, or locally with the command above.
